# Quality Checklist

Validate the result before finishing.

| Check | Pass Criteria |
|---|---|
| Core fields | All 8 core fields are present before strategy begins |
| Product clarity | The product is visually and strategically clear |
| Audience clarity | The target audience is visible through character and scene design |
| Consumer problem | The first panels clearly establish the problem |
| Product intervention | The product appears as the trigger, not just as decoration |
| Transformation | The change from before to after is obvious |
| Board title / headline | The title or headline appears clearly in TOP or BOTTOM |
| Key visual | The top section has a memorable transformation moment |
| 10 panels | There are exactly 10 storyboard panels |
| Packshot | The bottom section includes a clear product packshot |
| Branding | Brand identity and tagline are visible |
| Color system | Before and after color states are clearly different |
| Shot variation | The panel sequence uses varied shot types |
| Layout control | TOP / MIDDLE / BOTTOM structure is obvious |
| Prompt quality | The final image prompt is in English and suitable for image generation |

## Failure handling

If any important item fails:
1. Stop if a core field is missing.
2. Ask only for missing core fields.
3. Revise the strategy or panel sequence if storytelling is weak.
4. Strengthen product interaction.
5. Tighten the prompt before image generation.
