# Product Visual Language

Select the visual language that matches the product category.

| Category | Visual Language |
|---|---|
| Food | texture, steam, gloss, bite, crumbs, melted detail |
| Beverage | liquid splash, condensation, transparency, ice, droplets, refreshment |
| Beauty | skin detail, glow, hydration, texture refinement, transformation |
| Fashion | silhouette, fabric movement, pose, attitude, styling |
| Travel | open space, sunlight, atmosphere, escape, scenery, breathing room |
| Technology | interface, clarity, speed, light, order, clean surfaces |
| Service | before-after workflow, simplification, emotional relief, organized scene |
| Other | infer the most relevant sensory language from the product function |

## Additional guidance

- If the product is drinkable, show product interaction clearly.
- If the product is held, include grip, packaging, and hand interaction.
- If the product is wearable, show silhouette and movement.
- Always keep branding and package recognition clear in the bottom product zone.
