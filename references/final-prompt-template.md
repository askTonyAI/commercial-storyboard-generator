# Final Prompt Template

Use this as the default structure for the final English image prompt.

## Template

1. Visual quality layer
   - masterpiece, best quality, ultra detailed, cinematic lighting, professional storyboard sheet, high-end editorial layout, commercial campaign board, agency-level presentation style, strong visual hierarchy, visual storytelling clarity

2. Global control panel
   - Target language
   - Brand
   - Product
   - Product category
   - Character type
   - Gender
   - Age range
   - Genre
   - Tone preset
   - Board title / main headline

3. Reference safety system
   - Use reference images only for mood, styling, outfit direction, and color palette.
   - Do not replicate any real person's face.
   - Create a completely new fictional character.
   - Keep product shape, packaging, label design, and branding as accurate as possible.

4. Smart story engine
   - Story structure
   - Core flow: problem -> trigger -> transformation -> payoff -> ending

5. Key idea
   - One-line concept based on the brief.

6. Board title placement
   - The board title / main headline must appear clearly in the TOP or BOTTOM section of the storyboard board.
   - Use the selected Text Language.
   - Keep it short and readable.
   - Treat it as the main communication line of the campaign board.
   - Do not replace it with a random tagline unless the user provides one.

7. Document structure
   - ONE single 16:9 cinematic storyboard sheet
   - TOP: key visual + board title / main headline
   - MIDDLE: exactly 10 panels story
   - BOTTOM: product + branding + tagline

8. Key visual
   - Most impactful transformation moment
   - strong lighting contrast
   - emotional peak
   - product interaction included

9. Story flow panels
   - Describe all 10 panels using the panel template.

10. Shot system
   - wide shot, medium shot, close-up, extreme close-up, POV shot, over-shoulder, dynamic angle

11. Performance arc
   - Before: weak / tired / stressed
   - After: energetic / confident / attractive
   - Rewrite these states to match the brief.

12. Product cinematic system
   - Apply the correct category-specific visual language.

13. Product shot
   - clean background, perfect lighting, premium composition, clear branding

14. Ending frame
   - character looks at camera or delivers final emotional resolution
   - include tagline or board title if the user did not provide a separate tagline

15. Color system
   - Before color state
   - Trigger color bridge
   - After color state

16. Output control
   - one single 16:9 storyboard image
   - exactly 10 panels
   - editorial commercial layout
   - clear hierarchy
   - readable board title / headline
   - no repetitive shots
   - no weak storytelling

## Negative prompt starter

Use as a base and adapt when needed:
- repetitive shots, weak storytelling, unclear branding, distorted logo, unreadable text, bad anatomy, inconsistent character, messy layout, random panels, product not visible, low-end design, missing headline, missing title

## Important note

Add a brief note when needed that brand text and logos may require post-production correction.
